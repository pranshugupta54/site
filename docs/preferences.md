# Pranshu — taste, opinions, likes/dislikes

## Identity / positioning
- "Builds **products people want**" — mainly **backend**, cares about **UI/UX**, a bit of frontend. **Not infra** (don't claim it).

## Design taste (he is very particular)
- Highly sensitive to **fonts, background color, and sizing**. Will reject things for "bad fonts / weird colors / vibe-coded / too big".
- Likes: **warm** (cream/off-white) backgrounds, refined type, **subtle motion** (scramble text, marker highlight, gentle hops), small "exceptional details", terminal/dev flavor.
- Dislikes: generic, fancy/designer-y, dark-and-boring, oversized type, anything that looks fake or templated.
- Reference sites he loved: **jhey.dev, paco.me, ma5a.com** (minimal + alive + personality).
- Loved anime.js-style text effects; the Tofu mascot must feel **alive** (real frame animation, not a static glyph).
